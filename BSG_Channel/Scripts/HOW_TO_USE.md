# Bible Story Garden — How to Make a Video

## Step-by-step every time:

1. Open the **BSG_Scripts** folder (you're in it now)
2. Open the script file you want (e.g. 01_Jonah_and_the_Whale.json)
3. Press **Cmd + A** to select everything
4. Press **Cmd + C** to copy
5. Go to Chrome → **localhost:5002**
6. Scroll down to the Scenes section
7. Click **📋 Import Script from Claude**
8. Press **Cmd + V** to paste
9. Click **Load Script**
10. Hit **🎬 Generate Video** and walk away!
