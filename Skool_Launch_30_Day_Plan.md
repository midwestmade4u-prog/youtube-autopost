## 3. Module Outline (the course inside the community)

Each module = ~45-min screen recording + downloadable templates. Drip weekly so members stay engaged 8 weeks minimum.

1. **The Faceless Operator's Mindset** — Why this isn't a side hustle, it's a media business. Portfolio thinking. The 3-channel rule.
2. **Picking a Niche That Actually Pays** — RPM data, saturation analysis, monetization-friendliness, the niche scoring framework. Walkthrough of how MZ was chosen.
3. **Setting Up the Channel** — Gmail-per-channel pattern, YouTube channel creation, branding minimums, OAuth token generation, common 401 traps.
4. **The Automation Stack** — Live walkthrough of the actual GH Actions workflow: Claude script gen → edge-tts → Pexels footage → ffmpeg → upload. Members get the YAML.
5. **Scripts That Don't Suck** — Claude prompt engineering, validator regexes, hook formulas (incl. the first-3-words payoff rule), dedup ledger.
6. **Posting Cadence & Optimization** — When/how often, A/B testing thumbnails, reading YouTube Analytics like an operator, the 7-day deep-dive ritual.
7. **Cross-Posting Multiplier** — Same video → TikTok + Shorts + Reels. Watermark rules, caption rework, platform-specific hooks.
8. **Scaling to 10 Channels** — How to go from 1 channel to a portfolio without 10x'ing your time. Template repos, shared infra, when to hire.

**Bonus module (added month 2):** *Long-form expansion* — turning a viral Shorts channel into a long-form revenue compounder. Tied to the MZ long-form work happening in parallel = real-time case study.
